# MSA
MSA 구조의 이커머스 프로젝트입니다.

<br>

## 📌주요 기능
#### 1. Spring Cloud Gateway
 - 토큰버킷 쓰로틀러 적용으로 비정상 트래픽 대응 가능
 - 스프링 시큐리티 적용으로 로드밸런싱 단계에서 보안 가능
#### 
#### 2. Eureka
 - 라운드로빈 형식의 동적 로드밸런싱 가능
#### 3. OAuth
 - 카카오톡 소셜 로그인 가능
#### 4. Kafka
 - 크래프트 프로토콜을 사용해 주키퍼없이 카프카 단독으로 클러스터링
 - 협력적 리밸런싱 사용으로 효율적인 리밸런싱 가능

<br>
